# causal-data-science

Comprehensive causal data science skill for AI agents. Combines statistical methodology, causal inference, and experiment interpretation into a single skill.

## What It Does

- Guides causal analysis from observational data (DiD, IV, RDD, matching)
- Provides proper statistical testing workflows with assumption checking
- Emphasizes effect sizes and confidence intervals over p-values
- Addresses seed sensitivity and reproducibility
