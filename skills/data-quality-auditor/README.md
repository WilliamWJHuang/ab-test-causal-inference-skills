# data-quality-auditor

Systematic data quality auditing skill for AI agents. Checks for selection bias, survivorship bias, missing data patterns, data leakage, outliers, and sample representativeness before any analysis.

## What It Does

- 9-step audit workflow from data overview to pre-modeling checklist
- Missing data mechanism assessment (MCAR/MAR/MNAR)
- Data leakage detection (temporal, target, train-test, group)
- Cross-validation readiness check
- Class imbalance, metric alignment, and interpretability guidance

## Usage

Activate before running experiments, causal analyses, or ML pipelines to ensure data integrity.
