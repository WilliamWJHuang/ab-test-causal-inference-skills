# adaptive-explainer

Communication adaptation layer for AI agent skills. Detects user expertise level from conversational signals and adjusts statistical/causal inference explanations accordingly.

## What It Does

- Identifies 4 expertise levels (beginner → expert) from natural language cues
- Translates technical jargon to the appropriate level
- Adapts in real-time as the conversation reveals more about the user's background
- Includes a glossary mapping technical terms to plain language

## Usage

This skill is always active alongside other skills in the collection. It acts as a filter between the agent's technical output and the user.
