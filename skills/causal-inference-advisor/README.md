# causal-inference-advisor

Expert causal inference skill for AI agents. Guides users through identification strategies, method selection, and assumption validation for causal claims from observational and experimental data.

## What It Does

- Routes to correct causal method (DiD, IV, RDD, matching, synthetic control)
- Enforces identification strategy documentation
- Requires refutation and sensitivity tests
- Covers staggered adoption, SUTVA/interference, and heterogeneous treatment effects
