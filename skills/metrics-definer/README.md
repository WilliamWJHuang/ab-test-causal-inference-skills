# metrics-definer

Metric definition skill for AI agents. Helps users define primary metrics (OEC), guardrail metrics, and secondary metrics for experiments and product analytics.

## What It Does

- Guides OEC (Overall Evaluation Criterion) selection
- Defines guardrail metrics to prevent unintended harm
- Aligns statistical metrics with business objectives
- Covers metric sensitivity, directionality, and tradeoffs
