# stats-reviewer

Statistical methodology review skill for AI agents. Audits analyses for correct test selection, assumption verification, effect size reporting, and reproducibility.

## What It Does

- Reviews statistical test selection and assumption checking
- Enforces effect size and confidence interval reporting
- Checks for multiple comparison corrections
- Validates regression assumptions (linearity, homoscedasticity, VIF)
- Assesses seed sensitivity and reproducibility
