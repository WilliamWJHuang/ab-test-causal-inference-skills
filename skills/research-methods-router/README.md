# research-methods-router

Entry-point routing skill for AI agents. Helps users figure out which specialized skill to use for their research question by asking clarifying questions and suggesting multi-skill workflows.

## What It Does

- Routes to 5 specialized skills based on user intent
- 3-step clarification workflow (goal → data state → methodology)
- Pre-built workflows for common scenarios (A/B test, causal analysis, result audit)
- Methodological awareness section for smart routing decisions
- Covers parametric/non-parametric, effect sizes, multiple comparisons, Bayesian vs frequentist

## Usage

Activate when the user asks a vague or general research/analysis question.
