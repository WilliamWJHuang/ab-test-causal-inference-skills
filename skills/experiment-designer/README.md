# experiment-designer

End-to-end experiment design skill for AI agents. Guides users through power analysis, randomization, metric definition, and analysis planning for A/B tests and controlled experiments.

## What It Does

- Formal power analysis with minimum detectable effect sizing
- Randomization strategy selection (simple, stratified, cluster, switchback)
- Variance reduction techniques (CUPED/CUPAC, blocking, paired designs)
- Pre-registration and pre-analysis plan enforcement
- SRM detection and sequential testing guidance
- Interference awareness for platform experiments
